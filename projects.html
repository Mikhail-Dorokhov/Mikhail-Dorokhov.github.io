Automating the lights on my mouse

I recently was forced to replace my mouse. After doing my research, I chose the Steelseries Rival 100 which has been perfect except for the fact that there is no button on the mouse to turn off the built in lights. As a way of teaching my self about the basics of automation, I am creating a C++ program that uses a shell script to control the mouse lights based on the time of day by writing to virtual Linux files. This project is still in progress but you can find it here.
